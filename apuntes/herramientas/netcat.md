
# Netcat

Netcat es una herramienta de red que permite la transferencia de datos a través de la red. Es una herramienta muy versátil y se puede utilizar para una variedad de tareas de red, como la transferencia de archivos, la creación de túneles de red y la escucha de puertos.

## Parametros

- **-n** : No resuelve DNS
- **-l** : Modo escucha
- **-p** : Puerto
- **-v** : Modo verbose
- **-e** : Ejecuta un comando

ejemplo:

```bash
nc -nlvp 443
```
